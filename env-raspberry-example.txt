# ============================================================================
# SPIEGAZIONE DEI PARAMETRI
# ============================================================================
#
# DATABASE_URL:
#   - fintrack_user: nome utente database (creato durante import)
#   - fintrack_secure_password_123: PASSWORD DA CAMBIARE!
#   - localhost: il database è sullo stesso Raspberry
#   - 5432: porta predefinita PostgreSQL
#   - fintrack: nome del database
#
# JWT_SECRET:
#   - Stringa segreta per generare i token di autenticazione
#   - CAMBIALA CON UNA STRINGA CASUALE LUNGA!
#   - Puoi generarla con: openssl rand -base64 32
#
# PORT:
#   - Porta su cui il backend sarà in ascolto
#   - Default: 4000
#
# NODE_ENV:
#   - production: per uso normale
#   - development: per sviluppo (più log, hot reload)
#
# ============================================================================
# GENERA PASSWORD SICURE
# ============================================================================
#
# Sul Raspberry, esegui questi comandi per generare password casuali:
